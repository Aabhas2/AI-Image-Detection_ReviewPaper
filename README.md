# Review of AI-Generated Image Detection Techniques

## Overview

This academic literature review explores modern techniques for detecting AI-generated and manipulated images.

The review examines CNN-based detectors, transformer-based approaches, frequency-domain methods, and forensic techniques used to identify synthetic media generated by modern image generation systems.

## Topics Covered

* CNN-based image detectors
* Transformer-based detection models
* Frequency-domain analysis
* GAN image detection
* Generalization challenges
* Robustness and adversarial concerns
* Future directions in synthetic media forensics

## Key Findings

* Many detectors struggle to generalize across unseen image generators.
* Shortcut learning remains a major challenge.
* Robustness and cross-domain performance are active research problems.
* Hybrid forensic approaches appear promising for future detection systems.
